# 前端项目说明

## 项目概述

这是数据模型管理平台的前端应用，基于 React + TypeScript + Vite + Ant Design 构建，UI风格参考 Palantir Foundry 设计。

## 技术要点

### API代理配置
前端开发服务器配置了代理，将 `/api` 请求代理到后端服务器。需要在 `vite.config.ts` 中配置正确的后端地址。

### 类型安全
所有API接口都有对应的TypeScript类型定义，确保类型安全。

### 错误处理
统一的错误处理机制，自动显示错误消息。

### 动态表单
实例管理使用动态表单生成技术，根据对象类型的属性定义自动生成对应的表单字段，支持多种数据类型。

### 分页查询
列表页面统一使用分页查询，支持自定义每页显示数量。

## 运行说明

1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 确保后端服务运行在 http://localhost:8080
4. 访问 http://localhost:3000

## 项目结构

```
frontend/
├── src/
│   ├── api/              # API客户端
│   ├── pages/            # 页面组件
│   │   ├── Home/         # 首页
│   │   ├── ObjectType/   # 对象类型管理
│   │   ├── LinkType/     # 链接类型管理
│   │   ├── Instance/     # 实例管理
│   │   ├── Query/        # 查询构建器

│   │   └── Function/     # 函数管理

├── package.json
├── tsconfig.json
├── vite.config.ts

```
